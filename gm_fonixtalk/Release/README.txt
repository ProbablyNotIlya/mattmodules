How to Install
-----------------------
Copy all contents from the directory 'Dependencies' into the root directory of Garry's Mod.
    Ex.: E:\Steam\steamapps\<your username>\garrysmod
    
Copy gm_fonixtalk.dll into the lua/includes/modules directory in the directory of Garry's Mod.
    Ex.: E:\Steam\steamapps\<your username>\garrysmod\garrysmod\lua\includes\modules
